# Ruins Updater

A simple strip to automatically update 1.6.4 templates from the runins mod to 1.12.2

Feel free to fork, create version for other game versions, or improve this if you like.

To use just run the script using Python 3, and add the name of the file or folder you want to convert; 
it can be used on a sinlge file or an entire directory tree full of files.  The result will be in a 
folder called "updater-out" (or you can specify a second parameter to specify an output folder).
